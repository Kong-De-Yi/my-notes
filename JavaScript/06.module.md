# 模块

## 本质

- 一个模块就是一个文件
- export: 标记当前模块中可被外部访问的变量和函数
- import: 从其他模块导入功能
- 浏览器中需要指定脚本类型

```typescript
// sayHi.js
export function sayHi(user) {
  alert(`Hello,${user}!`);
}

// main.js
import { sayHi } from "./sayHi.js";

alert(sayHi);
sayHi("John");
```

```html
<!--script标签中指定脚本类型-->
<!doctype html>
<script type="module">
  import { sayHi } from "./say.js";

  document.body.innerHTML = sayHi("John");
</script>
```

## 功能

- 始终使用"use strict"
- 模块级作用域: 不同模块中的顶级作用域变量互不干涉
- 模块代码仅在第一次导入时被解析
- 顶层模块代码应用于初始化特定的内部数据结构，需要被多次调用的东西应该以函数的形式导出

```html
<script type="module">
  a = 5;<!--报错-->
</script>
```

```javascript
// alert.js
alert("Module is evaluated!");

// 1.js
import "./alert.js"; // Module is evaluated!

// 2.js
import "./alert.js"; // (什么也不显示)
```
